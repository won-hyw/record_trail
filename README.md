# 2022ITSHOW : 레코드 트레일
산책의 여유를 사진과 함께 기록할 수 있는 익명기록장

> 주요기능
- Android 개발 환경에서 Java를 사용하여 개발
- 전국길관광정보표준데이터 Open API를 이용하여 데이터를 파싱하여 지도와 연결
- 구글 지도를 이용하여 안내 길을 표시
